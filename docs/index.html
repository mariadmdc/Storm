About Storm

Storm is a high-level, object-oriented programming language designed specifically for young children who are learning how to code. Created by a team of computer science students at Loyola Marymount University, Storm aims to make programming accessible and enjoyable for beginners.

The language features a consistent and readable syntax that emphasizes clarity. Storm is statically typed, requiring users to declare variables, functions, and classes. To declare a new instance of anything, simply state set... (similar to var in other languages). We require the user to explicitly state the ending of loops, functions, or classes, making code structure clear and easy to understand.

Storm introduces intuitive terminology like "laps" instead of "for loops" and simplifies input with the "ask" command, avoiding the complexity of input streams found in other high-level languages. These design choices make Storm an ideal first programming language that builds a solid foundation for future learning.

Code Examples

1. Basic Program (Function) Example

# program (function) example - pizza counter
set program countSlices(people)
    set slicesEach to 2
    set totalSlicesNeeded to people * slicesEach
    say "You need " + totalSlicesNeeded + " slices of pizza for " + people + " people!"
end program

countSlices(10)   # Call the function for 10 people
countSlices(20)   # Call the function for 20 people

2. Loop Example

# infinite lap (loop) example
set laps to 0
start lap forever
    set laps to laps + 1
    say "You are on lap " + laps
    if laps = 4
        say "You finished 4 laps! You completed the mile!"
        stop running
    end if
lap

3. User Input Example

set pi to 3.14
set r to ask "What is the radius of your circle"

set program circle(radius)
    say "Your circle's area is " + pi*radius^2 + "."
    say "Your circle's circumference is " + 2*pi*radius + "."
end program

circle(r)

4. Object-Oriented Example

set object Animal()

    set program traits(species,name,personality)
        set Animal.species to species
        set Animal.name to name
        set Animal.personality to personality
    end program

    set program getInformation()
        say Animal.name + " is a " + Animal.species + " and is very " + Animal.personality
    end program

end object

Doggy = Animal(species to "dog", name to "Max", personality to "playful")
Doggy.getInformation()

5. Recursion Example

set a to 10

set program fibonacci(n)
    if n < 1 or n = 1
        return n
    otherwise
        return fibonacci(n-1) + fibonacci(n-2)
        #recurse like Python
end program

say "the " + a + " fibonacci number is " + fibonacci(a)
